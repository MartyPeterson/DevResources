Elements HTML Element Reference
----------------
<!DOCTYPE html> - defines the document type as HTML
<head> - contains metadata/information for the document
<title> - defines a title for the document
<body> - defines the document's body
<h1> to <h6> - defines HTML headings
<p> - defines a paragraph
<br> - inserts a single line break
<hr> - horizontal rule defines a thematic change in the content
<!-- Comments --> - defines HTML comments

Pick and Mix Selectors
.class - select a class
#id - select an id
* - select all
element - select an element ex: p {}
element, element - select multiple elements ex: h1, h2 {}
element.element - element with class selector ex: p.big {}
element element - descendant selector ex: article p {}
element > element - child selector ex: article > p {}
element + element - adjacent sibling selector
element ~ element - general sibling seletor

JS Conditionals
-----------------
if
else
else if
ternary operator
switch

if ex:
if (answer === true){
console.log("you said true!");
}

else ex:
if (answer === true){
console.log("you said true!");
} else {
console.log("you said something else");
}

else if ex:
if (answer === "Yes"){
console.log("you said yes!");
} else if (answer === "Maybe"){
console.log("you said maybe");
} else if (answer === "No"){
console.log("you said no");
} else {
console.log("you said something else");
}

terse if one line if statement ex:
if (cherub === "Cupid") console.log("Ouch an arrow!");

terse if else ex:
if (cherub === "Cupid") console.log("Ouch an arrow!");
else console.log("I feel nothing.");

ternary operator ex:
animal === "Cat"
? console.log("You will be a cat herder.") // if true do what is between ? and :
: console.log("You will be a dog catcher."); // if false do what is after the :

switch ex:
switch (answer) {
case "Yes":
console.log("you said yes!");
break;
case "Maybe":
console.log("you said maybe");
break;
case "No":
console.log("you said no");
break;
default:
console.log("you said something else");
break;
}

switch can continue until a break is hit cascading until the following cases
